# algorithm

## git사용법
- cd github (폴더 이용)
- git init(초기화 - 이 폴더를 git 저장소로 쓰겠다)
- git clone https://github.com/eunseo2/algorithm.git (깃허브에 있는거 가져오기)
- ls (복사한 저장소 폴더보기)
- cd algorithm
- git status (상태 보기)
- git add 파일이름
- git add * (내가 수정한 파일 다 넣고 싶을때)
- git commit -m "... "
- git push origin master
